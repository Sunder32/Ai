# MVP-11: MySQL, Быстрый старт, Бенчмарки и Чат с ИИ

## Описания того как вы поняли задачу
Перевести проект на базу данных MySQL, чтобы все работало более надежно. ТАк же теперь проект можно запустить — буквально одной кнопкой.

## Что было реализовано? Что было не реализованная?

**Реализовано:**
1.  **База данных MySQL**: Теперь проект работает на MySQL.
2.  **Быстрый запуск**: Сделал файл `start_ai_project.bat`. Теперь просто кликаешь по нему, и он сам все делает: проверяет питон, ставит библиотеки, создает базу и запускает сервер.
3.  **Чат с ИИ**: Теперь на сайте есть чат, где можно обсудить свою сборку с нейросетью.
4.  **Бенчмарки**: Добавили сервис, который показывает производительность процессоров и видеокарт в цифрах.
5.  **Профиль**: Страницу профиля переделали полностью, теперь она удобная.

**Не реализовано:**
- Пока база бенчмарков не очень большая, нужно будет наполнять данными.

## Выявленные баги
- Могут быть мелкие косяки с версткой на очень широких или узких экранах.

## Предложения по апгрейду, фиксам, фичам, ремувам
- Было бы круто добавить красивые графики производительности прямо в диалог с ИИ.
- Нужно залить больше данных по бенчмаркам для старого железа.
